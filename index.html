Interactive and visualized test model creation

In this tool you are able upload sample input data for a function you want to test as a CSV or JSON file. The uploaded data will apear as a table. Upon this file you can dynamically create a modell and later you may generate a test suite out of it.

Futhermore you will be able to make utilize the following features:

  • displaying combinatorical coverage
    • simple t-way coverage
    • total t-way coverage
    • ...
  • Improving of metrics by suggested additional tests out of generated input data
